변형과 모션

transform

  • 레이아웃에는 변화가 없는 상태에서 렌더링에 변화를 준다.
  • transform: scale(2) rotate(45deg) translate()
  • transform, opacity프로퍼티가 적용되면 z-index가 적용됨
  • transform의 %기준은 해당요소의 크기 기준

transition

  • css 프로퍼티가 변하는 시간을 설정
  • transition: 프로퍼티 또는 all 시간s
  • transition은 숫자로 변환이 가능한 속성에만 적용
  • transition-delay: 프로퍼티간의 transition 시작 시간 지연
  • transition-delay효과를 구현하려면 all을 사용하지 말 것
