CSS Cascading Style Sheets

우선순위와 상속이 존재하는 디자인 양식.

Type선택자 (태그선택자)

사용법

태그명{속성:값} 형태로 선언

특징

  • 동일한 태그에 모두 적용 된다.
  • 자식태그선택자가 우선순위가 높다.
  • agent style과 겹칠 수 있다.

Class 선택자

사용법

.클래스명{속성:속성값;}을 이용해서 선언 후, 클래스속성을 이용해서 호출

특징

  • Type 선택자보다 우선순위가 높다.
  • 반복호출이 가능하다.
  • 수정관리가 용이하다.

ID 선택자

사용법

#아이디명{속성:속성값} 선언 후, id속성을 이용해서 호출

특징

  • Class 선택자보다 우선순위가 높다.
  • 반복호출이 허락되지 않으며, 수정관리가 어려움.
  • html문법이나 프로그램용도로 사용.

InLine Style (인라인 스타일)

사용법

요소의 style속성을 지정 후 css를 작성

특징

  • ID 선택자보다 우선순위가 높다.
  • 스크립트를 이용해 css를 제어하는 데 사용.

    • CSS의 특징

    • 요소단위로 css가 적용된다.
    • 우선순위 : InLine Style, ID, Class, 자식태그, 부모태그
    • 글자정렬, 글자관련 스타일은 상속이 된다.

    가상클래스

    css rule에서 정의된 클래스명 ex):hover :active 등
