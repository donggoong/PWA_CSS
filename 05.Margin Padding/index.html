margin padding

  • margin(바깥쪽간격), padding(안쪽여백)
  • margin, padding : 위 오른쪽 아래쪽 왼쪽
  • margin, padding : 상하 좌우
  • margin : 전체
  • 첫번째 block요소는 margin-top 안됨,
    마지막 block요소는 margin-top 안됨,
    예외 (부모요소의 border나 padding이 적용될 경우)
  • padding과 border는 기본적으로 크기의 바깥쪽으로 적용됨,
    box-sizing속성을 이용해서 크기의 안쪽으로 적용할 수 있음
  • block요소간의 상하마진은 큰값만 적용 된다.(겹친다.)
  • Inline Level은 상하마진과 상하패딩이 적용되지 않는다.
